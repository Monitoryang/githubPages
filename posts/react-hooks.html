React Hooks 深度解析

2024-01-15 5 分钟阅读 1,234 次阅读

什么是 React Hooks？

React Hooks 是 React 16.8 引入的新特性，它让你在不编写 class 的情况下使用 state 以及其他的 React 特性。 Hooks 的出现彻底改变了我们编写 React 组件的方式，让代码更加简洁和可复用。

useState - 状态管理

useState 是最基础的 Hook，用于在函数组件中添加状态： 


import { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    
    return (
        <button onClick={() => setCount(count + 1)}>
            点击了 {count} 次
        </button>
    );
}

useEffect - 副作用处理

useEffect 用于处理副作用，如数据获取、订阅、DOM 操作等： 


import { useState, useEffect } from 'react';

function DataFetcher() {
    const [data, setData] = useState(null);
    
    useEffect(() => {
        // 组件挂载时获取数据
        fetch('/api/data')
            .then(res => res.json())
            .then(data => setData(data));
        
        // 清理函数（组件卸载时执行）
        return () => {
            console.log('组件即将卸载');
        };
    }, []); // 空依赖数组 = 只在挂载时执行一次
    
    return <div>{JSON.stringify(data)}</div>;
}

💡 小贴士： useEffect 的依赖数组非常重要，它决定了 effect 何时重新执行。空数组 [] 表示只在挂载时执行一次。

自定义 Hooks

你可以创建自己的 Hooks 来复用状态逻辑： 


// 自定义 useLocalStorage Hook
function useLocalStorage(key, initialValue) {
    const [value, setValue] = useState(() => {
        const stored = localStorage.getItem(key);
        return stored ? JSON.parse(stored) : initialValue;
    });
    
    useEffect(() => {
        localStorage.setItem(key, JSON.stringify(value));
    }, [key, value]);
    
    return [value, setValue];
}

// 使用
function App() {
    const [name, setName] = useLocalStorage('name', '访客');
    return <input value={name} onChange={e => setName(e.target.value)} />;
}

总结

React Hooks 让函数组件拥有了类组件的能力，同时保持了函数的简洁性。 掌握常用的 Hooks（useState、useEffect、useContext、useRef 等）是成为 React 高手的关键。
